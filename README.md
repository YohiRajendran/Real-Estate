Project Overview
This project analyzes real estate property listings and sales data using Power BI. The goal is to extract insights related to pricing behavior, market performance, property characteristics, time-based trends, and agent efficiency.
Business Objective
- Analyze real estate sales performance
- Understand pricing trends and valuation accuracy
- Identify factors affecting property value
- Evaluate market efficiency and agent performance
Dataset Description
Source: sale_price_real_estate.xlsx
Records: Approximately 2000 property listings
Each row represents a single property listing or sale.
Tools & Technologies Used
Power BI Desktop
Power Query
DAX (Data Analysis Expressions)
Microsoft Excel
Data Preparation
- Data cleaning and transformation using Power Query
- Handling missing and inconsistent values
- Creation of a Date Table for time intelligence
- Optimized data model for reporting
Report Pages
1. Summary
2. Raw data Analysis
3. Pricing & market Behavior
4. Property Analysis
5. Date-wise Trend Analysis
6. QA and Decomposition Tree

Visuals Used
KPI Cards, Column Charts, Line Charts, Donut Charts, Funnel Chart, Gauge Chart, Matrix, Decomposition Tree, Q&A Visual
Conclusion
This project demonstrates how Power BI can be used to transform raw real estate data into actionable insights that support data-driven decision-making.
